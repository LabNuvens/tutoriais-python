SIPAM Manaus S-Band Radar CAPPI Volumes v2.0a

Overview:

S-band radar volumes from Manaus, Brazil were shared by SIPAM for the time period beginning in January 2014 to December 2015 coinciding with the GOAmazon field campaign. The corrected radar reflectivity and uncorrected radial velocity fields from each volume were interpolated to a fixed grid. This is version 2.0a of the dataset, which benefits from improved quality control procedures to remove non-meteorological echo.
